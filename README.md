# README

这是一个常用函数的集合，包含了一些在数据展示场景下常用的一下方法，可能不适用于所有人。

## 安装

```shell
npm i data_tools_js
```

## 使用

### import

```javascript
import { funcName } from 'data_tools_js'
```

### require

```javascript
const { funcName } = require('data_tools_js')
```

### script

```html
<script src='xxx/data_tools_js' />
```

```javascript
data_tools_js.funcName
```
